# 园区指数接口文档

 utf-8 编码， get 请求 ，返回数据json格式

## 招聘指数

### 招聘指数-主要行业

行业不止一下两个，这为示例 实际行业为固定7个

```json
{
  "MainIndustry": {
    "制造业": "123",
    "金融业": "321"
  }
}
```

### 招聘指数-主要行业-时间

获取最近10个月份的主要行业(目前固定7个) 数据

```json
{
  "timeline": {
    "2016-11": {
      "MainIndustry": {
        "制造业": "123",
        "金融业": "321"
      }
    },
    "2016-10": {
      "MainIndustry": {
        "制造业": "123",
        "金融业": "321"
      }
    }
  }
}

```



### 招聘指数-7大类细分行业

GeneralIndustry  一共有7个行业大类

每类去取Top10的细分行业倒序排列

```json
{
  "SubdivideIndustry": {
```
